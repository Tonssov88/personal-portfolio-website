3-DOF Robotic Arm with Inverse Kinematics

Jan. 2026
C++ Arduino PCA9685 PWM Driver Inverse Kinematics

Designed and built a 3-degree-of-freedom robotic arm using servo motors, controlled via Arduino and PCA9685 PWM driver. Implemented analytical inverse kinematics algorithms to enable precise end-effector positioning.

Technical Challenges Solved:

  • Implemented analytical inverse kinematics to compute joint angles from Cartesian target coordinates
  • Utilized linear interpolation to generate smooth motion between start and end positions in 3D space
  • Developed C++ control software with enforced joint constraints for stable and safe motion
  • Integrated PCA9685 PWM driver for precise servo control with Arduino Nano microcontroller
  • Designed mechanical structure ensuring sufficient range of motion while maintaining structural integrity
